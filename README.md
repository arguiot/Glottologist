## Install


- Browsers:

```html
<script src="https://unpkg.com/glottologist"></script>
<script>
  const glot = new Glottologist();

  /* Your code */

</script>
```
- NodeJS:

```bash
npm i glottologist --save
```
```js
const glot = require("glottologist");
```
